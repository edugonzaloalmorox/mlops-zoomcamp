# MLops project 

This repo contains the materials for the mlops zoomcamp project. The project aims to build a ML tool that enables better searches within the contracts released by the Spanish public institutions. For that it will use Semantic Searches. 

# The problem

Currently the searches are limited to few fields which are constrained to the exact word(s) in the object  of the contract. This clearly limits the recognition and automation of processes.

# Getting the data

The ETL to retrieve the data consists of the  next process. 

+ Retrieve relevant information from the last updates associated with the contractors found in [this link](https://www.hacienda.gob.es/es-ES/GobiernoAbierto/Datos%20Abiertos/Paginas/LicitacionesContratante.aspx)
